## Study Method

### Weekly Inspiration Routine

```markdown
Monday: Browse 1 gallery (30 min)
- Screenshot 3-5 inspiring designs
- Note what makes each special

Wednesday: Deep dive (45 min)
- Pick 1 design from Monday
- Analyze using SPECS framework
- Document findings

Friday: Apply (60 min)
- Try to recreate one element
- Apply style to your project
```
